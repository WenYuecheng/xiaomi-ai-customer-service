# 小米智能客服机器人 - 项目计划与架构确认

## 1. 技术栈确认

### 1.1 前端技术
- **框架**: Vue 3 + Vite
- **UI 组件库**: Element Plus (表单、表格、上传、消息提示)
- **状态管理**: Pinia (登录和会话状态)
- **网络请求**: Axios (REST API)
- **内容渲染**: markdown-it + highlight.js (Markdown 与代码高亮)
- **流式输出**: SSE (Server-Sent Events，服务端向浏览器推送答案)

### 1.2 后端与 AI 技术
- **Web 框架**: Python 3.11 + FastAPI (统一 API 与 Swagger)
- **数据校验**: Pydantic
- **ORM 与数据库**: SQLAlchemy + SQLite (用户、文档元数据、会话、反馈持久化)
- **认证授权**: JWT + Bcrypt (登录、密码和角色权限)
- **RAG 框架**: LangChain (检索、Prompt、生成与会话编排)
- **Embedding 模型**: bge-small-zh-v1.5 (中文文本向量化)
- **向量数据库**: Chroma (向量持久化与相似度检索)
- **文档解析**: pypdf + python-docx (PDF/DOCX/TXT/MD 处理)
- **测试**: pytest + TestClient

### 1.3 部署与工程工具
- **容器化**: Docker Compose (前端、后端、持久化目录一键部署)
- **版本控制**: Git
- **项目管理**: Jira Software Scrum
- **环境变量**: .env + YAML
