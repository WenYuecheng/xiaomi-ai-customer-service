# 课程演示脚本

1. 在登录页切换“创建账号”，注册普通用户并自动进入动态 AI 数字空间。
2. 展示主题头像、咨询统计、兴趣星系、14 天趋势和活动时间轴；修改显示名称后刷新验证持久化。
3. 使用管理员、运营、普通用户账号分别登录，展示 RBAC。
4. 运营创建知识库，上传 PDF/DOCX/TXT/MD，查看任务状态、切分片段与重建。
5. 普通用户提出带型号问题，展示流式回答与真实来源卡片；再问无关问题展示兜底。
6. 使用 AI 选购生成方案，再从个人主页活动时间轴恢复问答和选购会话。
7. 连续兜底或直接要求人工，创建结构化工单；查询明确标记的 Mock 订单物流。
8. 运营查看日志、反馈、日/周/月热词并触发推荐训练。
9. 重启 Compose，再次登录新账号并打开个人主页、历史会话、文档和训练版本，证明持久化。
