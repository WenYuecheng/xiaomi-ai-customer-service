# 近两年产品参数知识库（待人工审核）

本目录包含 50 个产品文档：手机 18、平板 12、手表/手环 12、家电 8。手机和平板占比最高，手表/手环次之，家电最少。

## 使用方式

1. 先打开 `产品来源与人工审核表.xlsx`，按“人工审核”列逐条确认。
2. 打开每个产品的 Markdown，补齐标记为“待人工复核”的项目。
3. 删除不适用于目标销售地区的版本参数。
4. 审核通过后，再把对应 Markdown 上传到项目知识库。

## 质量边界

- 手机/平板以爱搞机文字资料为主；手表/手环以 IT之家为主；家电以小米官网和官方支持页为主。
- 没有下载完整网页或图片；只整理参数、文字结论和来源 URL。
- 英文官网参数已经翻译成中文。
- 独立实测与厂商标称不得混写。
- 所有文件当前状态均为“待人工审核”，不建议直接作为生产知识库。
